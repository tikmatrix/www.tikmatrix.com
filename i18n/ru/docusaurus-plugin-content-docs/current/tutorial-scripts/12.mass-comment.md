# Массовые комментарии

:::warning
Эта функция объединена в [Super Marketing скрипт](./14.super-marketing.md). Используйте Super Marketing для более мощных и гибких возможностей массовой рассылки и управления вовлечённостью.
:::

Массовые комментарии используются для размещения взаимодействий под большим количеством постов TikTok — подходит для развития аккаунтов, промо-кампаний и повышения вовлечённости. Система поддерживает заранее подготовленные шаблоны и генерацию в реальном времени, а также имитирует человеческую частоту действий, чтобы снизить риск детекции.

## Использование в Super Marketing

1. Откройте скрипт Super Marketing.
2. Выберите тип источника данных: «Список URL постов».
3. Вставьте ссылки на посты или импортируйте `.txt` (по одной ссылке на строку). После импорта можно выбрать "Добавить" (Append) или "Заменить" (Replace).
4. Выберите стратегию использования данных:

   - Общий пул: все боты берут цели из одного общего списка.

   - Однократное потребление: каждый бот обрабатывает уникальные цели, так что каждый пост получает комментарий только от одного аккаунта.

5. Установите, сколько комментариев публикуется на один пост (обычно 1 чтобы избежать дублей).
6. Настройте параметры комментариев:

   - Предварительно подготовленные шаблоны: вставьте несколько строк или абзацев готовых текстов. Каждая строка/абзац считается отдельным шаблоном.

   - Порядок комментариев: выберите «Случайный» или «По порядку».

   - Случайный: при отправке выбирается случайный шаблон.

   - По порядку: шаблоны используются в порядке вставки/импорта (удобно при необходимости воспроизводимой стратегии дедупликации).

   - Вставка эмодзи: включите автоматическое добавление эмодзи для большей естественности.

   - Вставка номера устройства (опционально): добавление номера устройства к комментарию для трассировки (используйте осторожно — в основном для отладки/аудита).

   - Генерация комментариев в реальном времени (опционально, ChatGPT API):

      - При включении скрипт может вызывать ChatGPT или похожий API для генерации контекстных и уникальных комментариев на основе заголовка/описания поста или доступного контекста — подходит, если нужно много уникального контента.

      - Внимание: для работы требуется валидный API-ключ; учитывайте ограничения по скорости, стоимость и требования по конфиденциальности (не отправляйте чувствительные данные сторонним сервисам).

   - Два распространённых способа обеспечить уникальность комментариев:

      1. Ротация заранее подготовленных шаблонов по порядку (каждый шаблон используется один раз до исчерпания）。

      2. Генерация уникальных комментариев в реальном времени с помощью ChatGPT。

7. Настройте прочие параметры скрипта (параллельность, интервал задач, пул аккаунтов и т.д.) и запустите скрипт.

:::tip Об интервалах задач
Установите минимальный и максимальный интервал и включите случайные задержки. Разнесённые по времени действия снижают риск одновременных пиков и ограничений со стороны платформы.
:::

## Рекомендации и примечания

- Тестируйте стратегию на небольшой выборке, чтобы проверить качество комментариев и реакцию платформы.
- Комментарии должны быть разнообразными и соответствовать правилам платформы; избегайте массовой отправки одинаковых сообщений.
- При использовании генерации в реальном времени контролируйте расход API и стоимость, не отправляйте конфиденциальные данные.
- Если часть целей не обработана, проверьте логи и повторно запустите обработку — уже выполненные строки будут пропущены.
