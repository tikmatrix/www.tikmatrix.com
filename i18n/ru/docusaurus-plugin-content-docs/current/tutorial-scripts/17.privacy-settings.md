# Настройки приватности

Скрипт **Настройки приватности** используется для массового обновления настроек приватности публикаций TikTok на основе фильтра по количеству просмотров.

:::info
Этот скрипт доступен только для аккаунтов TikTok.
:::

## Шаги

1. Выберите устройство для запуска скрипта.
2. Нажмите `Скрипты` > `Настройки приватности`.
3. Настройте параметры задачи:
    - **Параметры фильтра публикаций**:
      - **Максимальное количество просмотров**: Введите максимальное количество просмотров для фильтрации публикаций. Будут обновлены только публикации с просмотрами ≤ этого числа.
      - **Максимальное количество публикаций для обработки**: Введите максимальное количество публикаций для обработки. Установите 0 для неограниченной обработки.
    - **Параметры настроек приватности**:
      - **Кто может смотреть это видео**: Выберите один вариант:
        - `Все`: Видео может смотреть любой пользователь
        - `Друзья`: Видео могут смотреть только друзья
        - `Только вы`: Видео можете смотреть только вы
    - **Параметры переключателей**:
      - **Разрешить комментарии**: Включить или отключить комментарии к публикации.
      - **Разрешить повторное использование контента**: Включить или отключить повторное использование контента, включая Дуэт, Сшивку, Стикеры и Добавление в историю.
4. Нажмите `Запустить скрипт` для начала выполнения.

## Как это работает

Скрипт следует этому рабочему процессу:

1. Переход на вкладку Видео
2. Сканирование публикаций и анализ количества просмотров
3. Поиск публикаций, соответствующих критериям фильтра (просмотры ≤ max_views)
4. Переход на страницу подробностей публикации
5. Нажатие кнопки "Privacy settings"
6. Ожидание появления диалогового окна настроек
7. Установка разрешений на просмотр согласно вашему выбору
8. Интеллектуальное переключение опции "Allow comments"
9. Интеллектуальное переключение опции "Allow reuse"
10. Возврат к списку и продолжение обработки следующей публикации
11. Повторение до обработки всех подходящих публикаций

## Примечания

- Скрипт настроек приватности может не всегда выполняться успешно; при необходимости повторите неудачные задачи.
- Скрипт обрабатывает только публикации, соответствующие фильтру по количеству просмотров.
- Изменения применяются немедленно и не могут быть отменены через скрипт.
- Убедитесь, что вы проверили настройки приватности перед запуском скрипта для нескольких публикаций.
