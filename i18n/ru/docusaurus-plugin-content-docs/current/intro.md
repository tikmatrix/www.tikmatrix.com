# Введение в руководство

Изучите **TikMatrix** и **IgMatrix** менее чем за 5 минут и узнайте об их ключевых возможностях.

## Введение в программное обеспечение

**TikMatrix**  это настольное приложение для ПК, которое автоматизирует задачи TikTok на Android-устройствах.  
**IgMatrix**  инструмент для автоматизации задач в Instagram на Android.

**Примечание:** **TikMatrix** и **IgMatrix** — это два отдельных приложения и не могут работать одновременно на одном компьютере.

## Особенности

- **Переключение между несколькими аккаунтами**: управление несколькими аккаунтами на одном устройстве с автоматической сменой при выполнении задач.
- **Поддержка клонированных приложений**: поддержка клонированных TikTok/Instagram приложений с кастомными package name, чтобы разные аккаунты могли использовать разные клоны и автоматически открывать целевое приложение.
- ~~**Автоматическая регистрация**~~ **[СНЯТО С ОБСЛУЖИВАНИЯ]**
- **Автоматический вход**: Instagram поддерживает автоматический вход с 2FA; TikTok может требовать ручного вмешательства при некоторых проверках (captcha, подтверждение по email).
- **Автозаполнение профиля**: автоматически задавать аватар, никнейм, описание и имя пользователя.
- **Автоматическое распознавание аккаунтов**: обнаружение и импорт аккаунтов одним кликом, уже залогиненных на телефоне.
- **Развитие аккаунта**: поиск по ключевым словам, просмотр видео, лайки, подписки, комментарии и добавление в избранное для взращивания аккаунта.
- **Автопостинг**: публикация видео или изображений (поддержка сортировки и пакетной отправки) с настраиваемой музыкой, заголовком, описанием, хештегами и ссылками на товары.
- **Автоудаление**: удаление неудачных/слабо показывающихся постов по правилам просмотров.
- **Супермаркетинг**: имитация человеческой навигации (просмотр профилей, постов, случайные подписки/лайки/комментарии/избранное/поделиться/личные сообщения) и опция использования OpenAI или совместимого API для генерации персонализированных сообщений/комментариев.
- **Пакетное продвижение (Live)**: массовые лайки и комментарии в прямых эфирах.
- **Сбор пользователей**: сбор пользователей по ключевым словам, спискам подписчиков или подписок.
- **Массовая взаимная подписка и приветственные сообщения**: массовая подписка на подписчиков и отправка приветственных сообщений.
- **Планирование задач с точностью до минуты**: поддержка повторов, интервалов исполнения и максимального числа повторов.
- **Многопользовательская параллельность**: один ПК может подключать и управлять множеством Android-устройств и выполнять задачи параллельно.
- **Зеркалирование экрана**: зеркалирование нескольких устройств на Windows / macOS с управлением мышью и клавиатурой.
- **Импорт / Экспорт аккаунтов**: пакетный импорт/экспорт в формате Excel.
- **Многоязычная поддержка**: Программа поддерживает английский, русский и китайский языки. Примечание: для оптимальной совместимости некоторые функции требуют, чтобы приложение TikTok/Instagram на вашем телефоне было установлено на английский язык.
- **Техническая поддержка**: предоставляем поддержку через тикеты и персональную помощь.

Базовые функции (управление устройствами, подключение, визуальное управление, логи и статистика и т.д.) бесплатны и открыты; возможности автоматизации предоставляют соответствующие квоты параллельности для каждого плана.

Если вам нужно сменить компьютер, пожалуйста, обратитесь к нашему Руководству по миграции лицензии: [Руководство по миграции лицензии](tutorial-basics/license-migration.md).

## Что вам нужно

### Android-телефоны

Вы можете использовать любые Android-устройства для работы с TikMatrix:

- **Реальные Android-устройства**
- **[Материнские Android-устройства](http://www.niaozun.shop?cid=934ec2fe)**

- **Эмуляторы**

Устройства должны иметь минимум API 21 (Android 5.0) и не требуют root. Включите USB-отладку (Developer Options). На некоторых моделях (например, Xiaomi) может потребоваться включить USB debugging (Security Settings).

### Компьютер

#### Windows: Windows 10 или выше

#### Mac: macOS 15.0 или выше (Apple Silicon и Intel)
