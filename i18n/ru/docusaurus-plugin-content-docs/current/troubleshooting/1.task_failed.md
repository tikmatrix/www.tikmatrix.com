# Шаги по устранению неполадок при сбое задачи

## Шаг 1: Проверьте состояние агента (если задача не запустилась)

1. В боковой панели выберите: General → Open Agent.
2. Убедитесь, что на телефоне открыто приложение TikMatrix.
3. Наблюдайте, появится ли статус успеха в течение 10 секунд.
4. Если агент показывает статус ошибки, выполните ручную инициализацию, описанную в руководстве Initialize Device: ../tutorial-basics/2.init-device.md

## Шаг 2: Проверьте конфигурацию приложения TikTok/Instagram (если скрипт некорректно открывает/закрывает приложение)

1. Если вы используете официальное приложение TikTok/Instagram, нажмите "Open TikTok" / "Open Instagram" и проверьте, открывается ли приложение на устройстве. Если нет, нажмите кнопку настроек в заголовке и выберите корректную версию TikTok (International / Asia). У Instagram только официальная версия.
2. Если вы используете клонированную версию TikTok/Instagram, убедитесь, что вы настроили имя пакета клонированного приложения в **Account List**. См. руководство по использованию клонированных приложений: ../tutorial-basics/10.clone-apps-support.md. Также убедитесь, что при запуске задач включён режим **Multi-account Mode**.

## Шаг 3: Проверьте языковые настройки TikTok/Instagram

В настоящее время поддерживается только английский язык для TikTok и Instagram. Если ваши приложения настроены не на английский, установите язык приложения на English следующим образом:

1. Откройте приложение TikTok на устройстве.
2. Перейдите в свой профиль.
3. Нажмите три точки в правом верхнем углу, чтобы открыть настройки.
4. Прокрутите до раздела "Language".
5. Выберите в качестве языка приложения "English".
6. Повторите те же шаги для Instagram.

## Шаг 4: Запустите тестовый скрипт

1. В боковой панели выберите: Scripts → Test Script.
2. Дождитесь завершения скрипта.
3. Если тест не прошёл, проверьте логи и попытайтесь устранить проблему. См. раздел "Known Issues" ниже для распространённых проблем.

## Шаг 5: Отправьте запрос в службу поддержки

Если предыдущие четыре шага не помогли выявить проблему, создайте заявку в поддержку. Мы автоматически соберём логи и поможем в устранении неполадок. Шаги:

1. Нажмите кнопку "Support" в заголовке.
2. Нажмите "Create Ticket".
3. Введите контактную информацию и описание проблемы. Для ускорения решения прикрепите запись экрана выполнения скрипта.
4. Выберите затронутые устройства (можно выбрать несколько). Это важно — мы можем получить логи только для выбранных устройств.
5. Нажмите "Submit".
6. Мы можем запросить дополнительную информацию, например XML UI для последнего экрана перед ошибкой: вручную перейдите на телефоне на последний экран до сбоя задачи, нажмите правую кнопку "Debug" на экране устройства, затем нажмите "Download XML". Отправьте загруженный XML и номер заявки на нашу почту поддержки: [support@tikmatrix.com](mailto:support@tikmatrix.com)
