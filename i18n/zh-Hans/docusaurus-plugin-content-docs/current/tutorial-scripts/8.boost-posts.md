# 提升帖子

提升帖子脚本用于通过点赞、评论、分享、收藏和查看来提升帖子互动。

## 步骤

1. 确保账号已添加到账号列表中。
2. 确保已创建分组并且设备已移动到该分组。
3. 选择要互动的设备。
4. 点击 `脚本` - `提升帖子` 按钮。
5. 输入目标帖子URL，每行一个。
6. 选择要互动的类型：点赞/分享/收藏/查看。
7. 点击 `开始` 按钮开始提升帖子任务。

## 注意事项

* 提升帖子脚本不是100%成功的，您可能需要重试失败的任务。
