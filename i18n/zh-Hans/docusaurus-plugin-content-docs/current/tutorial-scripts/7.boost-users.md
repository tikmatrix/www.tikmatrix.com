# 提升用户

:::warning 下线提示
该功能已整合到 [超级营销脚本](./14.super-marketing.md)。请优先使用“超级营销”以获得更强大、更灵活的用户提升能力。
:::

提升用户功能通过批量关注/取关操作来提升目标用户的曝光与互动。系统会尽量模拟真实用户行为以降低被检测风险，但仍建议遵循平台规则并控制频率。

## 在“超级营销”脚本中使用提升用户功能

1. 打开脚本：选择“超级营销”。
2. 选择数据源类型为“用户账号列表”。
3. 批量粘贴用户名或从文本文件导入（每行一个）；导入后可选择“追加”或“覆盖”。
4. 选择数据使用策略：

    - 共享池策略：所有 bot 账号会对相同目标进行关注/取关。

    - 单次消耗策略：每个 bot 账号仅处理不同的目标，确保每个目标只被一个账号操作一次。

5. 在“用户操作 — 重复次数”中设置每个目标的操作次数（通常设置为 1 次）。
6. 选择操作类型：关注 或 取消关注。可在“用户操作”中进一步配置：

        - 进入用户主页方式：选择“搜索用户”或“直接打开主页（Direct Open Profile）”。

        - 发送私信（可选）：启用后可粘贴预生成的多行私信文案，每行或按段为一条候选文案；配置文案使用顺序：

                - 随机：每次使用随机选一条文案。
                - 按顺序：按粘贴/导入的顺序依次使用（可跨任务轮次继续）。

            另可选择是否在文案中自动插入表情（emoji）以提高自然度。

7. 配置其他参数（并发、延时、账号池），确认无误后启动脚本。

## 使用建议与注意事项

- 建议先在少量账号上测试，观察策略与参数是否触发平台风控。
- 合理设置延时与并发，模拟人工行为以降低风险。
- 长期运行时请定期检查账号健康、任务日志，并根据结果调整策略。
- 提升用户操作并非 100% 成功，遇到失败请重试或排查目标账号状态。
