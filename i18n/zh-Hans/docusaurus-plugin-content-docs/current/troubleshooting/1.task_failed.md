# 任务执行失败排查步骤

## 第1步：检查Agent状态(如果任务未能成功启动)

1. 点击左侧边栏“通用” - “打开Agent”
2. 确认手机是否已打开TikMatrix app
3. 观察是否在10秒内显示成功状态
4. 如果Agent状态为失败，请按照[初始化设备](../tutorial-basics/2.init-device.md)文档步骤进行手动初始化

## 第2步：检查TikTok/Instagram应用状态(如果脚本没有正确打开或关闭应用)

1. 如果您使用的是官方的TikTok/Instagram应用，请点击“打开TikTok”/“打开Instagram”，观察手机是否正确打开TikTok或Instagram应用,如果没有，请点击标题栏中的设置按钮, 选择正确的TikTok 版本(国际版/亚洲版), Instagram只有唯一的官方版本
2. 如果您使用的是克隆版TikTok/Instagram应用，请确保您已经在**账号列表**中正确配置了克隆应用的**包名**, 请参考[使用克隆版TikTok/Instagram应用](../tutorial-basics/10.clone-apps-support.md)文档进行配置, 并且确保运行任务的时候开启了**多账号模式**

## 第3步：检查TikTok/Instagram语言设置

目前我们只支持英文的TikTok和Instagram。如果您的TikTok或Instagram应用不是英文，请按照以下步骤进行设置：

1. 打开手机上的TikTok应用
2. 进入个人资料
3. 点击右上角的三个点打开设置
4. 滚动到“语言”部分
5. 选择“英语”作为语言
6. 重复以上步骤为Instagram应用设置语言为英语

## 第4步：运行测试脚本

1. 点击左侧边栏“脚本” - “测试脚本”
2. 等待脚本运行完成
3. 如果测试脚本运行失败，请查看日志并尝试解决问题，常见问题请参考下方的“已知问题”部分

## 第5步：提交工单

如果前4步检查都没有发现问题，需要创建一个工单, 我们会自动收集日志并协助您解决问题。请按照以下步骤操作：

1. 点击标题栏的“支持”按钮
2. 点击“创建工单”按钮
3. 输入您的联系方式和问题描述, 为了更快地解决问题, 请尽量拍摄脚本执行过程中手机的屏幕录像并添加到附件中
4. 选择受影响的设备,可以多选(重要, 只有选择了受影响的设备, 我们才能收到相关日志)
5. 点击“提交”按钮
6. 我们可能会进一步联系你以获取更多信息,比如下载指定页面的UI xml文件: 手动操作手机停留在任务失败前的最后页面, 点击手机大屏右侧的“调试”按钮, 然后点击“下载XML”按钮, 将下载的xml文件发送到我们的邮箱:`support@tikmatrix.com`并附上工单号
