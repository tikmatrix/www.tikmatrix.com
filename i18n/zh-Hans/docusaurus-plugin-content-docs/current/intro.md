# 教程介绍

在不到 5 分钟内了解 **TikMatrix** 和 **IgMatrix** 的核心功能与使用方法。

## 软件介绍

**TikMatrix** 是一款在安卓设备上自动执行 TikTok 任务的 PC 桌面软件。  
**IgMatrix** 是一款在安卓设备上自动执行 Instagram 任务的 PC 桌面软件。

**注意：** **TikMatrix** 和 **IgMatrix** 是两个独立的软件，不能在同一台电脑上同时运行。

## 功能特性

- **多账户切换**：同一设备可登录多个 TikTok/Instagram 账号并自动切换执行任务。
- **克隆 APP 支持**：支持使用自定义包名的 TikTok/Instagram 克隆应用，不同账号可绑定不同克隆应用并自动打开目标 APP。
- ~~**自动注册**~~：**[已下线]**
- **自动登录**：Instagram 支持 2FA 自动登录；TikTok 在部分验证（如图形验证、邮箱验证）上仍需人工介入。
- **自动填写资料**：支持设置头像、昵称、简介和用户名。
- **自动识别**：一键识别并导入手机上已登录的 TikTok/Instagram 账号。
- **自动养号**：支持搜索关键词、观看视频、点赞、关注、评论与收藏等养号行为。
- **自动发布**：支持发布视频或图片（可设置排序与批量数量），支持自定义音乐，并包含标题、描述、话题标签与商品链接。
- **自动删除**：根据播放量或指定规则删除效果不佳的帖子。
- **超级营销**：模拟真人访问路径（访问主页、浏览帖子、随机关注/点赞/评论/收藏/分享/私信），支持使用 OpenAI 或兼容 API 生成个性化私信与评论内容。
- **批量助推（直播）**：为直播批量点赞与评论。
- **采集用户**：通过关键词、粉丝或关注列表采集目标用户名。
- **批量回关**：批量关注粉丝并发送欢迎私信。
- **任务调度**：分钟级精度调度，支持自动重试、间隔执行与最大重试次数设置。
- **多设备并行**：一台电脑可同时连接并管理多部安卓设备并行运行任务。
- **屏幕镜像**：支持将多部设备屏幕镜像到 Windows / macOS，并通过鼠标与键盘进行控制。
- **导入 / 导出账号**：支持 Excel 格式的批量导入与导出。
- **多语言支持**：电脑软件支持英语、俄语与中文。注意：部分功能要求手机上的 TikTok/Instagram 应用语言为英语以获得最佳兼容性。
- **工单支持**：提供人工工单服务以解决使用过程中的问题。

## 你需要什么

### 安卓手机

你可以使用任何安卓设备来运行TikMatrix，包括：

- **真实安卓设备**
- **[主板安卓设备](http://www.niaozun.shop?cid=934ec2fe)**
- **[云手机](https://www.geelark.cn?invite_code=XHY6a8)**
- **模拟器**
- **虚拟机**

Android 设备至少需 API 21（Android 5.0），且不需要 root 权限。请在设备上启用 USB 调试（开发者选项）。在部分机型（例如小米）上，可能还需启用“USB 调试（安全设置）”以支持键盘与鼠标控制。

### 电脑

#### Windows：Windows 10或更高版本

#### Mac：macOS 15.0或更高版本（支持Apple Silicon M系列芯片和Intel芯片）

## 安装

1. 从[下载TikMatrix](https://tikmatrix.com/Download) 或者 [下载IgMatrix](https://igmatrix.com/Download-IgMatrix)页面下载最新安装程序。
2. 运行安装程序。
3. 等待安装完成。
4. 在你的电脑上打开**TikMatrix** 或者 **IgMatrix**应用程序。
