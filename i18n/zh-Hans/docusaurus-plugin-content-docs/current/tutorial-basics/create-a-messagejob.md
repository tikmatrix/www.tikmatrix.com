# 创建私信任务

私信任务用于自动给目标用户发送私信。

## 步骤

1. 点击设置按钮,配置私信目标用户的txt文件路径和需要发送的私信内容并点击保存按钮。
2. 选择需要执行私信的设备。
3. 点击发送私信按钮。
4. 点击私信任务检查任务是否创建成功。
5. 任务将在1分钟内开始执行。

- 私信内容可以配置多个，每行一个, 每个任务随机使用一个内容。
- 目标用户的txt文件内容是tiktok用户名,必须@开头,每行一个。
- 选择多个设备时, 会平均分配私信任务。
