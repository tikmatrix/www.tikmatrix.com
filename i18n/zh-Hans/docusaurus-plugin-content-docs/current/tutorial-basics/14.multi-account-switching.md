# 多账号切换

TikTok 和 Instagram 都支持在同一设备上登录多个账号，这是官方应用的原生功能。TikMatrix 和 IgMatrix 提供了自动切换账号功能，可以在所有已登录账号上自动执行任务，无需手动干预。

## 支持的脚本

以下脚本支持自动切换账号功能：

- **Profile（查看资料）** — 查看账号资料信息
- **Account Warmup（账号预热）** — 通过自然活动预热账号
- **Post（发布内容）** — 发布帖子
- **Mass DM（批量私信）** — 批量发送私信
- **Follow Back（回关）** — 回关关注你的用户
- **Unfollow All（取消全部关注）** — 取消关注所有账号
- **Delete Post（删除帖子）** — 删除已发布的内容
- **Boost Users（推广用户）** — 提高用户主页的互动量
- **Boost Posts（推广帖子）** — 提高帖子的互动量
- **Boost Lives（推广直播）** — 提高直播间的互动量
- **Mass Comment（批量评论）** — 批量发布评论
- **Boost Comments（推广评论）** — 提高评论的互动量
- **Super Marketing（超级营销）** — 高级营销自动化

## 初始设置：匹配账号

在使用多账号切换功能之前，必须先从设备中匹配账号。

1. 打开左侧导航中的 **脚本** 标签页。
2. 找到并运行 **匹配账号** 脚本。
3. 该脚本会扫描设备上已登录的账号，并自动将其添加到账号列表中。
4. 脚本执行完成后，点击 **账号** 按钮，验证设备上的账号是否已成功添加。

## 测试账号切换

运行完 **匹配账号** 脚本后，可以测试账号切换功能：

1. 在脚本标签页中运行 **切换账号** 脚本。
2. 该脚本没有其他任何逻辑——它的唯一作用是验证账号切换流程是否正常工作。
3. 如果脚本执行成功，说明账号切换功能已准备就绪。

## 使用多账号模式

完成初始设置和测试后：

1. 在运行任何支持的脚本时，在脚本设置中启用 **多账号模式**。
2. 系统会为账号列表中的每个账号创建独立的任务。
3. 任务会排队并依次执行——系统会自动切换到目标账号并运行自动化任务。
4. 无需任何手动干预。

## 何时重新运行匹配账号

**匹配账号** 和 **切换账号** 脚本通常只需要在初始设置时运行一次。但是，在以下情况下应重新运行 **匹配账号**：

- 在设备上登录了新账号
- 注销了现有账号
- 设备上的账号发生了变更

刷新账号列表的步骤：

1. 在 **账号** 页面手动清空账号列表。
2. 重新运行 **匹配账号** 脚本，检测设备上的当前账号。

---

多账号切换功能消除了手动切换账号的需要，让你可以轻松地在单个设备上的所有账号之间扩展自动化。
