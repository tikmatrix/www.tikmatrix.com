# 创建分享任务

分享任务是操作TikTok矩阵的多个账号对某个TikTok视频进行分享/转发的任务。

## 步骤

1. 选中需要执行分享任务的设备
2. 点击`Tk 工具箱` - `开始分享`按钮
3. 输入需要分享的tiktok视频链接,格式为`https://www.tiktok.com/@xxx/video/xxx`,可以从Tiktok网页版复制浏览器地址栏的视频链接
4. 点击`开始分享`按钮
5. 点击`任务` - `分享任务` 查看任务是否创建成功
